Projects

Blackjack

I developed a fully playable Blackjack game for my Grade 11 final project, a major accomplishment for me that was really exciting to build. Using Object-Oriented Programming by setting up classes like "Deck" and "Player" to manage game elements really helped this game come to life.

Personal Budget Tracker

This a budget tracker I built so I could manage my personal finances! It lets you log transactions, see your spending by category and it lets you keep an eye on your balance. All of your data saves automatically so you don't lose anything!

To-Do List

I built this simple to-do list because I wanted to practice and learn how to build a clean GUI. With this to-do list you can add tasks, mark them as completed, and delete them. Everything automatically saves so your tasks are there when you re-open the app!

Quiz!

This is a multiple choice quiz app I built using tkinter. This quiz game features a 30 Second timer for each question, visual feedback showing if the answer is incorrect or correct, and score tracking! The questions shuffle after every playthrough, covering a variety of topics.
